Privacy Policy

IFC IDS Creator

IFC IDS Creator is designed as a local-first application for creating, reviewing, importing and exporting Information Delivery Specification files.

No tracking or advertising

Summary

IFC IDS Creator does not require an account.

The app does not contain advertising or third-party analytics.

The app does not sell personal information and does not use project data for advertising or tracking.

Information collected by the app

IFC IDS Creator does not intentionally collect or transmit personal data to the developer.

The app does not collect:

  • Account or login information
  • Advertising identifiers
  • Usage analytics
  • Location data
  • Contacts
  • Browsing history
  • Payment information
  • Project files for developer-controlled cloud storage
  • AI prompts or generated-response transcripts

Local project storage

IDS projects and drafts are stored locally within the app's sandbox on the device.

When an existing IDS file is imported, the app may preserve a local copy of the original source so it can later be exported exactly. The user can remove this preserved source using the app's forget-source action.

Deleting the app may remove locally stored drafts and preserved files, subject to normal operating-system backup and restoration behaviour.

File import, export and sharing

The app accesses files only after the user selects an import, export or share action using Apple's system interfaces.

The app itself does not upload IDS files to a developer-operated server.

If a user chooses iCloud Drive or another third-party file provider in Apple's Files interface, that provider processes the selected file under its own terms and privacy policy. IFC IDS Creator does not control those external services.

Apple Intelligence

On compatible devices and operating systems, the app can use Apple's on-device Foundation Models framework for optional drafting, suggestions, explanations and summaries.

The app does not use a third-party AI provider and does not require a developer-managed API key.

Only the structured context needed for the requested task is supplied to the Apple system language model. The app excludes original imported file bytes, complete raw XML, security-scoped file URLs and unrelated internal project notes from AI prompts.

AI output is temporary until reviewed. Only structured IDS rules approved by the user become part of the saved project.

Apple Intelligence availability and operating-system behaviour are controlled by Apple and the user's device settings.

Reports

PDF, Markdown and JSON reports are generated locally from the structured IDS project and deterministic review results.

Reports do not intentionally include sandbox paths, security-scoped URLs, original imported XML bytes, AI prompts or hidden diagnostic data.

Analytics, advertising and tracking

IFC IDS Creator does not include:

  • Third-party analytics SDKs
  • Advertising SDKs
  • Cross-app tracking
  • Third-party AI SDKs
  • Developer-operated cloud processing

System diagnostics

Apple may provide operating-system diagnostics, crash reporting or App Store information according to the user's device and Apple account settings. These services are controlled by Apple and are not operated by IFC IDS Creator.

Children's privacy

IFC IDS Creator is a professional construction and BIM productivity application. It is not directed specifically at children and does not provide social, advertising or account features.
