IFC IDS Creator

Create, edit, review, import and export Information Delivery Specification files on iPhone, iPad and Mac.

Getting started

  1. Open IFC IDS Creator and choose Create New IDS.
  2. Enter the IDS title, purpose, project information and target IFC versions.
  3. Add one or more specifications.
  4. Define which IFC objects each specification applies to.
  5. Add the properties, attributes, classifications, materials or relationships that are required.
  6. Open Review to check deterministic errors and warnings.
  7. Use Preview to inspect the generated IDS XML.
  8. Use Files to export the completed .ids document.

Templates

The app includes editable starter templates for common information requirements, including model information, doors, spaces, classifications, manufacturer data, assets and FM handover.

Templates are starting points. They should be reviewed and adapted to suit the project, appointment, exchange information requirements and applicable standards.

Importing an IDS file

Open Files and choose Import IDS. Select a local .ids document using the system file picker.

The app will review the document before replacing the current workspace. Unsupported or future content is clearly identified before conversion. The original imported file can also be retained locally for exact source export.

Review and export

The deterministic Review screen identifies missing or conflicting authoring information before IDS export.

  • IDS export: exports the structured .ids file.
  • PDF Summary: creates a professional human-readable report.
  • Markdown Summary: creates a portable text report.
  • JSON Summary: creates a structured report for other workflows.

Report exports are summaries and do not replace the standards-based .ids document.

Apple Intelligence

On supported devices and operating systems, optional Apple Intelligence features can draft requirements, suggest improvements, explain rules, explain review issues and summarise the project.

Generated content is always presented for review. AI does not export XML directly and does not change the project until the user approves the structured proposal.

When Apple Intelligence is unavailable, all manual authoring, templates, review, import and export features remain available.

Troubleshooting

An IDS file cannot be exported

Open Review and resolve any blocking errors. Human-readable reports can still be exported while a draft contains findings.

An imported file needs conversion

The file contains content the visual editor cannot preserve exactly. Review the listed unsupported content before opening a converted copy. The original source can remain stored locally.

Apple Intelligence is unavailable

Apple Intelligence requires a supported device, operating system and enabled system model. Continue using the manual builder and templates, or check the device's Apple Intelligence settings.

The system save or share panel does not show “.ids”

Apple system panels may display only the filename stem or a generic file description. The exported file itself retains the correct .ids extension.

Important distinction

IFC IDS Creator is an IDS authoring application. It creates and reviews the requirements document itself.

It does not validate an IFC model against an IDS. Use a dedicated IFC IDS validation application for model checking.
